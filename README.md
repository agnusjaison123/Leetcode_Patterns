# Leetcode_Patterns
## Two Pointers
### The decision path to Two Pointers:
- Is the input a sorted array or string? → Yes
- Are you searching for a pair/triplet or a subarray? → Yes
- Do you need O(1) extra space? → Yes → Two Pointers
